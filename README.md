### Projects

**[UniKit AI](https://github.com/NintendaDev/unikit-ai)**<br>
A spec-driven AI framework for Unity development. Provides a full development cycle - from idea exploration to code review and documentation - without writing complex prompts. Features dynamic memory that loads stack-specific rules (UniTask, R3, Zenject, etc.) by task context, and a self-learning system that turns bug fixes and code reviews into improved project rules.

**[Modular Unity Game Template](https://github.com/NintendaDev/modular-unity-game-template)**<br>
A production-oriented Unity template built around a state machine for game lifecycle management. Modular architecture with Zenject, Addressables, and UniTask. Includes examples of analytics, ads, localization, and save system integration.
